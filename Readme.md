## How to Run

1. **Clone the Repository:**

    ```sh
    git clone https://github.com/yourusername/advent-of-code-2021-golang.git
    cd advent-of-code-2021-golang
    ```

2. **Navigate to the Day's Challenge:**

    ```sh
    cd day1
    ```

3. **Run the Solution:**

    ```sh
    go run part.go
    ```
